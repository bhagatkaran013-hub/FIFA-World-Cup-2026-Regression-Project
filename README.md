# ⚽ FIFA World Cup 2026 Player Performance Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting the **Performance Score** of FIFA World Cup 2026 players using Machine Learning Regression techniques. The workflow includes data preprocessing, Exploratory Data Analysis (EDA), feature engineering, model building, evaluation, and comparison of multiple regression algorithms.

The objective is to identify the factors that influence player performance and develop predictive models capable of estimating a player's overall performance score based on match statistics and player attributes.

---

## 🎯 Project Objectives

- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Visualize player performance trends.
- Encode categorical variables.
- Train multiple regression models.
- Evaluate model performance using regression metrics.
- Compare different machine learning algorithms.

---

## 📂 Dataset Features

The dataset contains information related to FIFA World Cup 2026 players, including:

- Player ID

- Nationality
- Team
- Position
- Preferred Foot
- Matches Played
- Goals
- Assists
- Pass Accuracy
- Tackles
- Saves
- Market Value
- Player Rating
- Performance Score (Target Variable)
